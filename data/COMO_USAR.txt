PROGRAMA PARA TRANSFORMAR EL ARCHIVO DE DATOS DE ALUMNADO AL FORMATO NECESARIO PARA LA BASE DE DATOS

**COMO USARLO**

-Se debe tener instalado previamente Java 8 o posterior antes de ejecutar el programa.
En caso negativo, se puede descargar desde esta dirección web: https://www.java.com/es/download/
-Una vez instalado Java 8, copiar el archivo de datos del alumnado dentro de la carpeta 'data'.
El archivo debe tener el nombre 'DNI.txt'.
-Los datos deben aparecer uno por linea del archivo, con el formato de:
	CLAVE, APELLIDOS, NOMBRE.
IMPORTANTE:El primer item debe ser la clave, y los tres datos deben estar separados por coma.
-Lanzar la aplicación 'dni_parser.exe'
-Aparecerá un archivo llamado 'database.json'. Este es el archivo que hay que importar a la base de datos.

-EN CASO DE PRODUCIRSE UN ERROR EN LA EJECUCIÓN DEL PROGRAMA, ENVIAR UN CORREO A
